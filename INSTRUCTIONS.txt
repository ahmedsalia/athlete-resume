=========================================================
  BASKETBALL ATHLETE RESUME — HOW TO USE
=========================================================

FILE LOCATION:
  athlete-resume/resume-template.html

HOW TO VIEW:
  Open resume-template.html in any web browser (Chrome, Edge, Firefox).

HOW TO PRINT / EXPORT TO PDF:
  1. Open the file in your browser
  2. Press Ctrl+P (Windows) or Cmd+P (Mac)
  3. Choose "Save as PDF" as the destination
  4. Set paper size to Letter (8.5 x 11 in)
  5. Set margins to None
  6. Click Save

---------------------------------------------------------
  WHAT TO REPLACE FOR EACH PLAYER
---------------------------------------------------------

All items marked with <!-- REPLACE --> in the HTML need
to be updated for each athlete. Here's a checklist:

IMAGES (place in the same folder as the HTML):
  [ ] logo.png         — Your team logo
  [ ] player-photo.jpg — The athlete's photo

PERSONAL INFO (sidebar):
  [ ] Age
  [ ] Height (e.g. 6'4" / 193 cm)
  [ ] Weight (e.g. 185 lbs / 84 kg)
  [ ] Nationality
  [ ] Canadian Citizenship (Oui/Non/Résident permanent)
  [ ] School name
  [ ] Position

PLAYER HEADER (right side):
  [ ] Player full name (PRÉNOM NOM)
  [ ] Position & jersey number
  [ ] Current team name

STATS (right side):
  [ ] Season (e.g. 2025-2026)
  [ ] PPG, RPG, APG, FG%, SPG (top row)
  [ ] 3PT%, FT%, BPG, TO, MIN (bottom row)

EXPERIENCE (right side):
  [ ] Team name, years, and league for each entry
  [ ] Add or remove <div class="experience-item"> blocks

SOCIAL MEDIA (sidebar):
  [ ] Instagram handle
  [ ] Twitter / X handle
  [ ] TikTok handle
  [ ] Add or remove social items as needed

CONTACT (sidebar):
  [ ] Email
  [ ] Phone number

HIGHLIGHTS:
  [ ] YouTube or video link URL

---------------------------------------------------------
  TIPS
---------------------------------------------------------

• To create a resume for a new player:
  1. Copy resume-template.html
  2. Rename it (e.g. john-doe-resume.html)
  3. Replace all the placeholder values
  4. Place the player photo in the same folder

• To add or remove stat boxes, copy/delete the
  <div class="stat-box"> blocks in the HTML.

• To add more experience entries, copy a
  <div class="experience-item"> block and edit it.

• The colors #0F3B2C (dark green) and #E5E4E2 (platinum)
  are set in the :root CSS variables at the top of the file.

• The page is bilingual (French/English labels).
  To switch to English only, edit the labels in the HTML.
